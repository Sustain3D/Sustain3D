In Practice: TREC Cooper Hawk Case Study

This page documents how the Sustain 3D workflow was applied in practice for the Tom Ridge Environmental Center (TREC) Cooper Hawk specimen.

1) Project snapshot

Project

TREC Specimen Scans (TREC_SpecimenScans_2026)

Artec Space Spider

Processing software

Naming basis

VAR Lab pattern TREC_XXXX_<SpecimenName>

Publication status

Access copy published; preservation package prepared for DOI workflow

2) How the workflow played out

Capture

Raw scans were created from the specimen

The object entered the project as structured-light scan data.
  1. TREC provided the specimen model for scanning.
  2. Raw scans and exported high-resolution OBJ were created from Artec Space Spider capture.

Processing

Raw and preservation packages were retained

Working files stayed in project folders while the master and access paths were separated.
  1. Raw and preservation packages were retained, and a smaller web-access model was derived.
  2. Blender working files were kept in project processing folders.

Publication

The access derivative was released separately

The public-facing model was treated as an access copy, not the preservation authority.
  1. Access GLB was published to Sketchfab.
  2. Preservation master package and metadata were copied into repository preservation storage for long-term retention.

Contained TREC_0001_CooperHawk_v01_pres/, the preservation-ready export prepared for handoff.

04_access/

Stored TREC_0001_CooperHawk_v01_access.glb, the smaller access derivative for publication.

documentation/

Held the metadata and paradata notes that explained how the case study package was created.

Raw package

This retained the original capture outputs as the closest record of the scanning event.

Preservation master package

This was the authoritative long-term package for the processed model.

Metadata file

The metadata note preserved context, rights, scale, and processing decisions so the package can be interpreted later.

This handoff kept working directories lean while preserving the full archival package in dedicated preservation storage.
