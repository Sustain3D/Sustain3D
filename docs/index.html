Practical 3D Data Preservation

Sustain 3D is a web resource for teams that create 3D data and need durable organization, metadata, storage planning, and policy decisions without a large dedicated preservation staff.

Disclaimer: any step can be changed to fit local needs. This project outlines basic information and a starting workflow. This project was created by Remington Orange as a partial requirement for fulfilling the requirements of completing a thesis with Schreyer's Honors College

Who this is for

  • University labs
    Teams generating scans that need a repeatable project structure.
  • Small museums
    Institutions digitizing collections with limited technical staffing.
  • Libraries and archives
    Programs adapting 2D preservation workflows to 3D packages.
  • Community heritage projects
    Groups preserving local or cultural materials with practical guardrails.

What this site does

1) Give one clear path

The goal is to provide one streamlined resource for understanding how to preserve 3D data from setup through long-term access.

2) Define actions at each step

Each page provides practical workflow steps, examples, and decision support for implementation.

3) Scales from institution to personal

This resource is designed to guide both small organizations and individual practitioners in better storing and managing their 3D data.

4) Teaches about 3D data

It explains core 3D preservation concepts, including structure, naming, metadata, storage, rights, and publication planning.

Core Idea

Use one repeatable structure, document what was done as work progresses, and publish access copies while preserving stable master files.
